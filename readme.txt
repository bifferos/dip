
Download iPlayer
================

License: Public domain.


Steps:

1) git clone https://github.com/dinkypumpkin/get_iplayer
2) Open a console and run ./server.py
3) In Chrome, install the extension in developer mode:
   - Menu -> More Tools -> Extensions
   - Check 'developer mode'
   - Load unpacked extension, select this directory
4) Visit the place you want to download something.
5) Click the 'DiP' icon at the top right
